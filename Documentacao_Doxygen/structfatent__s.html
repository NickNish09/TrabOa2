Detailed Description

Structs utilizadas para simular a tabela fat na memória.

Fatent: utilizada para mapear cada setor do HD, indicando se o mesmo está sendo utilizado e o próximo setor relacionado. Fatlist: utilizada para mostrar o nome do arquivo e o primeiro setor por ele ocupado no HD.
